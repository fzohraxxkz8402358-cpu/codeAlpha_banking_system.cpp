# Banking Management System in C++

## 📌 Project Overview

The **Banking Management System** is a console-based C++ application designed to simulate basic banking operations. It allows users to create and manage bank accounts, perform transactions, and store account information using file handling. This project demonstrates core C++ programming concepts, modular programming, and data persistence.

---

## ✨ Features

* Create New Bank Account
* Deposit Money
* Withdraw Money
* Check Account Balance
* Search Account by Account Number
* Display All Customer Accounts
* Update Customer Information
* Delete Customer Account
* File Handling for Permanent Data Storage
* Menu-Driven User Interface
